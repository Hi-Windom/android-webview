## 用法

1. 从 release 下载安装包
2. 在安卓设备上安装
3. 重启系统

实测 vivo oppo 联想可以安装。

小米等品牌的系统无法安装，这个没办法。

## 来源

从 https://www.apkmirror.com/ 下载 arm64-v8a + armeabi-v7a 非捆绑 apk
