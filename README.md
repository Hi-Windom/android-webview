## 用法

> 如果 Github 下载困难，也可以使用 [阿里云盘](https://www.alipan.com/s/biwoA42wv7v) 下载，提取码: 20pj

1. 从 release 下载安装包
2. 在安卓设备上安装
3. 重启系统

> [!TIP]
> 实测 vivo oppo 联想可以直接安装。

> [!NOTE]
> 小米系统无法直接安装 webview 更新。需要先在开发者选项中临时关闭系统优化（原 MIUI 优化），安装 webview 更新后再打开系统优化即可。

## 回退

在软件管理界面卸载 webview 更新即可
